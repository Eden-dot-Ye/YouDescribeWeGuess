# 项目目标

我想要做一个web网站，使得可以创建房间，有主持人界面（可以随机选人和揭示答案）和普通观众界面，并能够给随机观众题目让其开麦描述，其它观众猜测，并且输入的猜测忽略大小写，支持模糊匹配，并且只要有人猜中就会全体人公示，并且有实时排行榜功能

## 项目名称

YouDescribeWeGuess

## 技术栈

- 框架：NextJS
- 数据库：Supabase

## 界面要求

1. 首页：展示游戏介绍，会显示加入房间和主持人登录的入口。
2. 房间界面：分为主持人界面和观众界面。
   - 主持人界面：可以随机选择房间中的人和揭示答案。
   - 观众界面：可以看到主持人给出的题目，并且可以输入猜测。
3. 猜测结果界面：当有人猜中时，公示答案和猜中的人，并展示排行榜。
4. 实时排行榜界面：在所有界面的右侧，实时展示所有参与者的得分情况。
5. 主持人设置题目界面：主持人可以添加、编辑和删除题目（题目包含中文和英文两个部分），还可以设置题目的时间限制（如果时间限制到期但没有人猜中，也会公示答案）。

整体界面采用现代化设计，简洁明了的同时，越好看越好。

## 功能需求

1. 主持人需要登录，登录用户名和密码默认admin/admin，登录后可以进入主持人创建房间和设置题目界面（分两步让主持人输入）。
2. 观众可以通过房间号加入房间，并参与猜题游戏。
3. 判定观众的猜测是否正确时，忽略中英文，忽略大小写，并且支持模糊匹配（例如：正确答案是“Apple/苹果”，观众输入“Apple”、“apple”或者“苹果手机”都算猜对）。
